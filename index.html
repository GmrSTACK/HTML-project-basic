O que é HTML

>>> (Voltar)

HTML é uma linguagem de marcação usada para estruturar o conteúdo da web. Ao contrário de linguagens de programação como Python ou JavaScript, que executam ações lógicas, o HTML é uma linguagem estática usada para definir a estrutura de uma página web. Ele utiliza tags (ou etiquetas) para indicar os diferentes tipos de conteúdo e como eles devem ser exibidos no navegador. HTML é interpretado pelos navegadores web, que convertem o código em uma página visual interativa. Cada página web é composta por um conjunto de elementos HTML que descrevem títulos, parágrafos, listas, imagens, links e outros componentes essenciais.

Estrutura Básica de um Documento HTML

>>> (Voltar)

A estrutura de um documento HTML é composta por uma série de elementos e tags (ou etiquetas) que ajudam a organizar e definir o conteúdo de uma página web. Um documento HTML básico possui algumas seções fundamentais, que são lidas e interpretadas pelos navegadores para exibir o conteúdo de forma adequada.

  • <!DOCTYPE html> - Esta linha é a primeira do documento e indica que a página é escrita em HTML5, a versão mais recente do HTML. O DOCTYPE informa ao navegador qual tipo de documento ele deve esperar, garantindo que o conteúdo seja renderizado corretamente.
  • <HTML> - é o elemento raiz de qualquer documento HTML. Ela engloba todo o conteúdo da página, exceto o DOCTYPE, que é especificado antes. O atributo lang="pt-br" indica que a língua principal do conteúdo é o português do Brasil, o que pode ser útil para otimizar a página para mecanismos de busca e acessibilidade.
  • <HEAD> - A seção <head> contém informações meta sobre a página, mas que não são exibidas diretamente na tela. Dentro do <head>, você geralmente encontrará:
    • <meta charset="UTF-8">; - Especifica o conjunto de caracteres utilizado na página, garantindo que ela exiba corretamente caracteres especiais (como acentos e cedilhas).
    • <meta name="viewport" content="width=device-width, initial-scale=1.0> - Essa meta tag ajuda a garantir que a página seja responsiva, ou seja, que se ajuste corretamente em dispositivos móveis.
    • <title> - Define o título da página, que aparece na aba do navegador. No exemplo, o título é "Exemplo de Página HTML".
  • <BODY> - O corpo do documento HTML é onde reside o conteúdo visível da página, ou seja, tudo o que o usuário verá ao acessar a página.
    • <p> - Define um parágrafo. Aqui, temos o texto "Este é um parágrafo de exemplo, escrito em HTML."
    • <a href="https://www.exemplo.com"> - A tag <a> define um link (âncora) para outra página ou recurso. O atributo href especifica o endereço de destino do link. No exemplo, ao clicar no link "Clique aqui para visitar o exemplo.com", o usuário será redirecionado para "https://www.exemplo.com".

Principais Elementos HTML

>>> (Voltar)

HTML é composto por uma variedade de elementos que ajudam a estruturar o conteúdo da página. Alguns dos elementos mais comuns incluem:

  1. Textos e cabeçalhos
    • <h1> a <h6> - Definem cabeçalhos, sendo <h1> o mais importante e <h6> o menos importante.
    • <p> - Define um parágrafo de texto.
    • <strong>- Define um texto em negrito, indicando ênfase importante.
    • <em> - Define um texto em itálico, geralmente usado para enfatizar.
  2. Listas
    • <ul> - Lista não ordenada (com marcadores).
    • <ol> - Lista ordenada (numerada).
    • <li> - : Item da lista.
  3. imagens
    • <a>: Define um link. O atributo href é usado para especificar o destino do link.
    • <img>: Define uma imagem. O atributo src especifica a origem da imagem.

História do HTML

>>> (Voltar)

O HTML (HyperText Markup Language) é a espinha dorsal da World Wide Web, sendo a linguagem fundamental para a criação de páginas e sites. Sua história é marcada por inovações, mudanças significativas e o crescimento explosivo da Internet. Vamos explorar a evolução do HTML e como ele moldou a maneira como interagimos com a web hoje.

  1. O Início da Web e o Surgimento do HTML (1989 - 1991)

    A história do HTML começa com Tim Berners-Lee, um cientista britânico que trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear). Em 1989, Berners-Lee propôs a ideia da World Wide Web, um sistema que permitiria compartilhar informações de maneira mais fácil através de links e documentos interligados. Para tornar essa ideia viável, ele desenvolveu três tecnologias essenciais: HTML, URI (Uniform Resource Identifier) e HTTP (HyperText Transfer Protocol).Em 1991, Tim Berners-Lee criou o primeiro site da web, no qual introduziu o conceito de páginas interligadas por meio de links. O HTML foi a linguagem de marcação utilizada para estruturar o conteúdo do site, permitindo que textos e imagens fossem organizados de forma legível e navegável. A primeira versão do HTML era bem simples e focada em marcar textos e links.

  2. HTML 2.0: Primeira Padronização (1995)

    Com o sucesso crescente da web, tornou-se necessário criar uma versão mais padronizada do HTML. Isso levou à criação do HTML 2.0, que foi publicado em novembro de 1995 pelo World Wide Web Consortium (W3C), organização fundada por Tim Berners-Lee para padronizar a web. O HTML 2.0 consolidou as especificações iniciais da linguagem e adicionou novos elementos, como formulários e tabelas, que passaram a ser utilizados amplamente.

  3. 4. HTML 4.01: A Consolidação e Adoção de Padrões (1999)

    Em 1999, o HTML 4.01 foi lançado, uma versão mais madura e estável da linguagem, que consolidou as versões anteriores e introduziu importantes mudanças para melhorar a acessibilidade e a separação entre estrutura e estilo. O HTML 4.01 também passou a incorporar o conceito de código semântico, que visava melhorar a leitura do código e a acessibilidade dos sites. Uma das grandes inovações foi a introdução da recomendação de usar CSS (Cascading Style Sheets) para controlar a aparência dos sites, ao invés de utilizar diretamente as tags de formatação dentro do HTML. Isso permitiu que o HTML fosse focado na estruturação de conteúdo, enquanto o CSS cuidava da apresentação.

  4. O HTML Hoje: Uma Linguagem em Evolução Constante

    Desde o lançamento do HTML5, a linguagem continua sendo aprimorada, com atualizações sendo feitas regularmente para introduzir novos recursos e melhorar a performance. Embora a maioria dos desenvolvedores tenha adotado amplamente o HTML5, a especificação continua sendo desenvolvida e aprimorada pelo W3C e pelo WHATWG. Atualmente, o HTML continua sendo a principal linguagem para a criação de conteúdo na web. Ele é a base para muitas outras tecnologias como CSS e JavaScript, que são usadas para melhorar a aparência e a interatividade das páginas.
